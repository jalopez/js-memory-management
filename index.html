Understanding memory managment

Basic concepts

  • Heap memory
  • Garbage collection
  • Pointer
  • Memory graph
"The heap area is the dynamic memory pool where objects are allocated when they are created by a program"
"The garbage collector is the process that frees a memory area when the objects allocated in it, are no longer retained (being used) by the program"
"A pointer is a reference to a memory position. This reference is stored in a variable and points to a value of a primitive type"
"Visual representation of how memory is organized, showing variable values starting in the root node until leaf nodes"

JS Primitive types

  • Number
  • Boolean
  • String
  • Object (including arrays and functions)
  • + DOM Nodes (only in browser)
